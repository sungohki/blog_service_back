# 블로그 서비스 용 백엔드 프로젝트

- 블로그 서비스를 운영하는데 있어 필요한 데이터베이스의 정보를 제공하는 API 서버 프로젝트입니다.
- Koa 라이브러리를 통해서 구현한 REST API 서버 프로젝트입니다.

## 구조

- src
  - api : api 경로 및 엔드 포인트 모음
  - middlewares : 전처리를 위한 미들웨어 함수 모음
  - models : mondo db 데이터베이스 모델 모음
  - constants : 상수값 정의 모음

## 기능

- 사용자 권한
  - 회원가입
  - 로그인
- 포스팅
  - 전체 조회
  - 세부 조회
  - 글 생성
  - 글 수정
  - 글 삭제
